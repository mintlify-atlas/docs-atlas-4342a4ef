The poll command allows server managers to create interactive polls that members can vote on using reactions. Polls support up to 10 options and can optionally be configured to end automatically after a specified time.

## Creating a Poll

```bash
/poll question:What should we do this weekend? option1:Gaming option2:Movie Night option3:Sports
```

<Note>
  Requires the **Manage Server** permission to create polls.
</Note>

## How It Works

1. **Poll Creation**: When you create a poll, Bastion posts an embed with your question and all the options
2. **Automatic Reactions**: Bastion automatically adds letter reactions (🇦, 🇧, 🇨, etc.) corresponding to each option
3. **Member Voting**: Server members vote by clicking on the reaction that matches their choice
4. **Poll Management**: The poll creator can end the poll manually using the "End Poll" button
5. **Auto-End**: If a timer is set, the poll will automatically end after the specified duration

## Poll Display

Polls are displayed as an embed with:
- **Author**: "POLL" label at the top
- **Title**: Your poll question
- **Fields**: Each option with its corresponding letter emoji (🇦, 🇧, 🇨, etc.)
- **Reactions**: Automatic letter reactions for voting
- **Button**: "End Poll" button for manual poll termination
- **Timestamp**: (Timed polls only) Shows when the poll will end

## Managing Polls

### Ending a Poll Manually

Click the **End Poll** button on the poll message. This button is only visible to members with the Manage Server permission.

### Viewing Results

To see poll results, check the reaction counts on the poll message. Each reaction shows how many members voted for that option.

## Premium Features

## Best Practices

- **Clear Questions**: Write clear, concise questions that are easy to understand
- **Distinct Options**: Make sure each option is clearly different from the others
- **Reasonable Duration**: Set appropriate timer durations (24-72 hours for most polls)
- **Option Limit**: While you can add up to 10 options, 3-5 options often work best for clarity
- **Pin Important Polls**: Consider pinning important polls so members can easily find them

## Limitations

- Minimum of 2 options required (option1 and option2)
- Maximum of 10 options
- Poll timer range: 1-720 hours (if using timed polls)
- Only members with Manage Server permission can create and end polls
- Timed poll limits apply based on your Premium tier

## Technical Details

Poll data is stored in the database with the following information:
- Message ID
- Channel ID
- Guild ID
- End time (for timed polls)

Timed polls automatically expire and are cleaned up 10 days after they end.
