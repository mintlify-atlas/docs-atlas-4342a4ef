Bastion provides powerful search capabilities across multiple platforms and databases. From web searches to entertainment databases, find information quickly without leaving Discord.

## /web

Search the web using AI-powered search with Tavily API.

<ParamField path="query" type="string" required>
  The search query you want to search for.
</ParamField>

<ParamField path="freshness" type="choice">
  Specifies the time range for search results.
  - `Past day` (d)
  - `Past week` (w)
  - `Past month` (m)
  - `Past year` (y)
</ParamField>

<ParamField path="topic" type="choice">
  Specifies the category of the search.
  - `General` - General web search
  - `News` - News articles
  - `Finance` - Financial information
</ParamField>

### Usage Examples

```bash
# Basic web search
/web query: What is the capital of France

# Recent news search
/web query: technology trends freshness: w topic: news

# Financial information
/web query: stock market trends freshness: d topic: finance
```

### Response

Provides:
- AI-generated answer summarizing results
- List of sources with titles and links
- Response time and number of sources

<Note>
  Requires `tavilyApiKey` in settings.yaml. Get your API key at [tavily.com](https://www.tavily.com/#pricing)
</Note>

  The search query for Wikipedia.
</ParamField>

### Usage Example

```bash
/wikipedia query: Discord
```

### Response

Returns:

- First 10 sentences of the article
- Automatically follows redirects

## /weather

Get current weather information for any location worldwide.

<ParamField path="location" type="string" required>
  The location for the weather forecast (city name, zip code, etc.).
</ParamField>

### Usage Examples

```bash
/weather location: London
/weather location: New York, NY
/weather location: 90210
```

### Response

Provides detailed weather information:
- Current condition with icon
- Temperature (Celsius and Fahrenheit)
- Feels like temperature
- High and low temperatures
- Cloudiness and humidity percentages
- Wind speed and direction
- Atmospheric pressure
- Visibility distance
- Sunrise and sunset times

<Note>
  Requires `openWeatherMapApiKey` in settings.yaml. Powered by OpenWeatherMap.
</Note>

## API Key Configuration

Many search commands require API keys. Configure them in your `settings.yaml`:

```yaml
# Search Commands
tavilyApiKey: "your_tavily_key"
openWeatherMapApiKey: "your_openweathermap_key"
tmdbApiKey: "your_tmdb_key"
wordnikApiKey: "your_wordnik_key"
nasaApiKey: "DEMO_KEY"
coinMarketCapApiKey: "your_coinmarketcap_key"

# Game search (IGDB via Twitch)
twitch:
  clientId: "your_client_id"
  clientSecret: "your_client_secret"
  accessToken: "your_access_token"
```

See the [API Keys Configuration](/configuration/api-keys) guide for detailed setup instructions.
